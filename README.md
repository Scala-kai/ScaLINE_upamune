# LINE ppoi yatsu (Oouso)

## Usage

### Commands

- create
  - create user
  - ex) ``create `id` `name` `tel` `email` ``
- change
  - change user
  - ex) ``change `id` ``
- send
  - send message your friend
  - ex) ``send `id` `text` ``
- list
  - list your friends
  - ex) `list`
- who
  - who am i ?
  - ex) `who`
- follow
  - follow user (you can use id or tel, email)
  - ex1) ``follow `id` ``
  - ex2) ``follow `tel` ``
  - ex3) ``follow `email` ``
